# HiTPoly
A platform for setting up high throughput polymer electrolyte MD simulations.

To run this code you need to download and install LigParGen locally on your machine via the tutorial [here](https://www.linkedin.com/pulse/how-install-ligpargen-server-locally-leela-sriram-dodda/)

Install HiTPoly via 
`
pip install -e .
`
Other packages that will be needed includes Packmol, rdkit, openbabel, scipy, torch, openmm.
